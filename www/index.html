发音生硬 / 有噪音 / 读不准？点这里（重要）

发音质量取决于手机的“文字转语音(TTS)引擎”。手机自带的引擎常常机械、有杂音、吞音。 按下面装一次 Google 语音引擎，就能变成自然、有语气的英美发音：

  1. 应用商店搜索并安装 “Speech Services by Google”（Google 文字转语音）。
  2. 手机 设置 → 系统 → 语言和输入法 → 文字转语音输出。
  3. 把“首选引擎”改成 Google。
  4. 点引擎旁的设置 → 安装语音数据 → 下载 English (United States) 或 English (United Kingdom)。 建议选带“高质量/High quality”的那一项。
  5. 回到本页选“美式/英式”，点“试听发音”确认效果。
